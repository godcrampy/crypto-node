# Crypto Node

Block-chain implementation to play around with written in Javascript.

Currently Implemented:

- sha256
- Commitment
- Proof of Work
- Workless Blockchain
- Blockchain

## Getting Started

- `$ git clone https://github.com/godcrampy/crypto-node`
- `$ yarn` (Optional. Have a look at _package.json_)
- Start Playing
